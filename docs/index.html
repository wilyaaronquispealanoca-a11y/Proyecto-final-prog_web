Simulador de Compras Familiares

Inicio

Esta página permite registrar productos por mes, calcular el gasto familiar, comparar aumentos de precios y visualizar cómo cambia el poder adquisitivo.

Objetivo

Analizar la evolución de los precios y el impacto en el presupuesto familiar.

Función principal

Registrar meses y productos para generar tablas y gráficos dinámicos.

Contexto

Una familia mantiene el mismo ingreso mensual, pero los precios cambian con el tiempo.

Cuando los precios aumentan, una familia puede comprar menos con el mismo ingreso. Este simulador ayuda a visualizar ese cambio de forma simple e interactiva.

Simulador

Agrega meses, productos y compara los resultados.

Acciones

Puedes agregar varios meses y, dentro de cada mes, varios productos con su precio y cantidad.

Resultados

Los cálculos se muestran dinámicamente en la página.

Interpretación del impacto de la variación de precios.

La variación de precios afecta directamente la capacidad de compra de una familia. Este simulador permite observar ese impacto mes a mes.
